Dank Dunks

Made with Unity.

Dank Dunks was created during cyber security classes. I did the final polish and added more levels after the classes.

The game is a 2D physics based puzzle game, where you launch the basketball player and try to score a dunk by rotating your ragdoll character.

The game is an omage to classic flash games with simple graphics but addicting gameplay ideas.

Key features
  • 2D ragdoll system
  • Can be played in a browser (WebGL)

PuckOff

Powered by Unreal Engine 4.

PuckOff is an ice hockey game developed for the PC.
The game is developed by Kuura Playhouse where I worked as the main programmer.

I was also in charge of making the music and handling all sound related stuff.

PuckOff is a 3rd person ice hockey game with a story mode emphasizing on character customization. (Appearance and stats)

Key features
  • Sub mesh based customization system
  • Ice hockey team AI
  • Split-screen multiplayer
  • Character skill tree system

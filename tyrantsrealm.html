Tyrant's Realm

Made with Unity.

Tyrant's Realm was co-created with artist Hermanni Penttala.
Tyrant's Realm is a short souls-like action game with a retro 3d aesthetic.

The game was developed in two months, with the bulk of the dev time used during the last month, when the game loop and the general direction of the game was decided. The first month was dedicated to creating the combat system and figuring out some designing stuff.

The game also includes an equipment system, with collectable and equippable weapons and armor. The player can equip these items in the inventory screen.

The game uses a save file system that keeps track of the player's equipment, and the loot in the game world.

It has a simple dialogue system and an action context popup.

I also created a "designer friendly" way to create new armor and weapons with different stats and 3d-models without the need for additional programming by using scriptable objects in Unity.

Key features
  • Souls-like combat
  • Collectable and equippable items
  • Persistent level loot
  • Enemy combat AI
  • Custom skybox/fog shader
